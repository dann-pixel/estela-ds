¿Qué es Estela DS? Sistema de diseño corporativo basado en Angular Material v20

Estela Design System es un tema de Angular Material v20 (Material Design 3) creado para estandarizar la interfaz de todos los productos digitales de Estela.

Provee un conjunto de tokens de color, tipografía y forma que se aplican globalmente a todos los componentes de Angular Material, sin necesidad de crear nuevos componentes desde cero.

Los equipos de desarrollo simplemente instalan el tema, lo configuran en styles.scss y todos los mat-button, mat-input, mat-card, etc. adquieren el look & feel de Estela automáticamente.

Instalación 4 pasos para tener el tema activo en tu proyecto
1

Instala el paquete

Instala directamente desde el repositorio de GitHub.

bash 
npm install git+https://github.com/dann-pixel/estela-ds.git

info Requiere Node.js ≥ 20 y Angular CLI ≥ 19. El paquete se instalará como estela-angular en tu node_modules.

2

Configura angular.json

Agrega el path de los archivos SCSS del DS al compilador de estilos.

angular.json 
"build": {
  "options": {
    "stylePreprocessorOptions": {
      "includePaths": [
        "node_modules/estela-angular/src"
      ]
    }
  }
}

info Agrégalo tanto en el builder de build como en el de test si usas estilos en tus specs.

3

Aplica el tema en styles.scss

Importa el DS y activa el tema claro. El setup emite dos bloques CSS para garantizar el orden correcto de los overrides.

styles.scss 
@use 'index' as estela;

// Tema claro (requerido)
@include estela.light-theme-setup();

// Tema oscuro (opcional)
// Tema oscuro (opcional) — activa .dark-theme en <html> desde TypeScript
// Tema oscuro (opcional) — activa .dark-theme en <html> desde TypeScript
@include estela.dark-theme-setup();

html, body {
  height: 100%;
  margin: 0;
}

body {
  font-family: var(--mat-sys-body-large-font), sans-serif;
}

h1, h2, h3, h4, h5, h6 {
  font-family: var(--mat-sys-headline-large-font), sans-serif;
}
4

¡Listo! Usa los componentes

Todos los componentes de Angular Material ya tendrán el tema de Estela aplicado.

any.component.html 
<!-- Los componentes de AM adoptan el tema automáticamente -->
<button mat-flat-button>Guardar</button>
<button mat-stroked-button>Cancelar</button>

<mat-form-field>
  <mat-label>Nombre</mat-label>
  <input matInput />
</mat-form-field>

<mat-card appearance="outlined">
  <mat-card-content>Contenido</mat-card-content>
</mat-card>

Clases utilitarias del DS Clases globales disponibles cuando se instala el tema

.form-field-sm

Form field condensed de 40px de alto. Ideal para barras de búsqueda y filtros alineados con botones.

<mat-form-field class="form-field-sm">
  <mat-icon matPrefix>search</mat-icon>
  <input matInput placeholder="Buscar..." />
</mat-form-field>

Chips de status semántico

Combina mat-chip con las clases .status-chip + estado para mostrar estados con color semántico en tablas.

 Personalización avanzada Cómo extender o modificar el tema desde tu proyecto

palette Cambiar colores de marca

Edita projects/estela/src/lib/theme/_palette.scss y actualiza los tokens de la marca.

// _palette.scss
$brand-primary:           #00b5cc;   // Color principal
$brand-primary-container: #bfecf2;   // Contenedor (botones tonal, chips)
$brand-on-primary:        #ffffff;   // Texto sobre primary

text_fields Cambiar tipografía

Edita projects/estela/src/lib/theme/_typography.scss y actualiza las Google Fonts en index.html.

// _typography.scss
$estela-brand-family: Outfit;            // Headings (sin comillas)
$estela-plain-family: 'Instrument Sans'; // Body y labels

dark_mode Activar dark mode

Agrega dark-theme-setup() en styles.scss e inyecta ThemeService en tu shell para alternar el tema.

// styles.scss
@include estela.dark-theme-setup(); // genera .dark-theme { ... }

// theme.service.ts — usa signal + localStorage + prefers-color-scheme
@Injectable({ providedIn: 'root' })
export class ThemeService {
  readonly isDark = signal<boolean>(/* detecta OS / localStorage */);
  toggle() { this.isDark.update(v => !v); } // aplica clase en <html>
}

// app.component.ts
readonly theme = inject(ThemeService);

// app.component.html
<button mat-icon-button (click)="theme.toggle()">
  <mat-icon>{{ theme.icon() }}</mat-icon>
</button>
